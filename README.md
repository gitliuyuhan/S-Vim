# S-Vim
# 方便于远程服务器上使用的vim配置

## 功能

1. vim配色(接近gvim,支持solarized,molokai配色)、终端命令提示符高亮(配置在bashrc中)
2. 语法高亮
3. pathogen/Vundle插件管理
4. C/C++代码补全，python补全，javascript/nodejs补全(配色)。YCM插件太大，所以用其他补全插件代替
5. python代码跳转
6. 文件浏览
7. tagBar函数、变量、类浏览

## 文件浏览NerdTree
- F4 &emsp;打开/关闭
- o键 &emsp;在当前窗口打开新文件
- i键 &emsp;分屏打开新文件
- 鼠标双击 &emsp;相当于按o键
- Ctrl-t &emsp;返回原文件

## 大纲式导航TagBar
- F3 &emsp;打开/关闭
- Ctrl-o &emsp;返回到上次位置
